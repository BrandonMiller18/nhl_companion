About the NHL Companion.

The NHL Companion started as a project for me to learn and have fun with. I had created a CLI app in Python to sound a horn every time the Blues scored - it was a way to feed my hockey addiction during COVID. Then, it became a poorly formatted website. Now it is this. In the future, it may be something else.

How it works.

The NHL Companion is built using Python. The site is built using the Python Flask webframework and Socket-IO to facilitate server to client communication. The data is retrieved from the NHL's public API. When a user selects a team, a session variable denoting the user's favorite team is set in the session. You'll notice this when you navigate back to the homepage after selecting a team, your team's game/standings will be highlighted and bold. This session variable is also used to display the team name in the header of the app page.

Once a user selects a team, they then select a "stream delay." This is tricky to get right, but once you figure it out for your specific use case, it generally stays the same. This is to facilitate the inconsistancy between someone who streams games using an app/website vs. someone who watches on cable. The stream delay is then sent to the server, which triggers the app to start running. If there is no game for that team, that is displayed to the user and they are redirected to the home page.
